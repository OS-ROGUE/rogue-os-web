# 🚀 Rogue OS - Official Landing Page

Welcome to the official web repository for **Rogue OS**, a high-performance Arch Linux-based distribution.

## 🛠 Project Goals
- **Performance First:** Built on Arch for absolute speed and control.
- **Neon Aesthetic:** A sleek, cyberpunk-inspired UI (Violet/Neon).
- **Gamer-Centric:** Optimized for the modern user.

## 💻 Tech Stack (Planned)
- **Framework:** React / Next.js
- **Styling:** Tailwind CSS + Framer Motion
- **Deployment:** Vercel
